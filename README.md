# dam-banking-system
# Sistema de gestión de cuentas bancarias

## Descripción

Sistema de gestión de cuentas bancarias que permite:

- Crear nuevas cuentas
- Eliminar cuentas existentes
- Ingresar dinero
- Retirar dinero
- Realizar transferencias entre cuentas
- Validar titular antes de operar

---

## Tecnologías utilizadas

- Java
- Programación Orientada a Objetos
- ArrayList
- Control de flujo con `switch`
- Validación de datos con `Scanner`

---

## Conceptos aplicados

- Encapsulamiento
- Modularidad
- Gestión de colecciones
- Validación de entradas del usuario
- Métodos personalizados (`ingresar`, `retirar`, `transferir`)
- Control de errores

---

## Estructura del proyecto

- `Principal.java` - Punto de entrada
- `Cuenta.java` - Modelo de datos
- Métodos de gestión bancaria y lógica del sistema

---

## Objetivo

El objetivo del proyecto es simular un sistema bancario básico aplicando buenas prácticas de programación en Java y reforzando los fundamentos de POO.

Aplicación desarrollada en Java como proyecto académico del ciclo DAM (Desarrollo de Aplicaciones Multiplataforma).
